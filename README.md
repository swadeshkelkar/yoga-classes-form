
# Yoga-classes-form

Registration form  for a yoga class


## To run app

1. Backend
 - Create a .env file in the backend folder and your MONGO_URI into it.

```bash
  cd backend
  npm install
  node server.js
```

2. Frontend
```bash
  cd frontend
  npm install
  npm start
```
